# feishu-task - 飞书任务管理

在飞书群聊或私聊中创建和管理任务，支持 @提及成员分配任务。

## 功能

- ✅ 创建任务（支持截止时间和提醒）
- 👥 群聊中 @提及成员分配任务
- 🔔 提前 N 分钟提醒
- ✅ 完成任务
- 📋 查询任务列表
- 🗑️ 删除任务

## 使用方式

### 私聊使用
直接发送命令给机器人

### 群聊使用
在群聊中 @机器人，然后发送命令

## 命令格式

### 创建任务（简化）
```
/task 创建任务 <任务标题>
```

### 创建任务（完整）
```
/task 创建 <标题> [截止时间] [--reminder 分钟数] [--desc 描述]
```

### 分配任务给群成员
```
/task 分配 <@成员> <标题> [截止时间]
```

### 完成任务
```
/task 完成 <任务ID>
```

### 查询任务
```
/task 列表
/task 查看 <任务ID>
```

### 删除任务
```
/task 删除 <任务ID>
```

## 示例

### 私聊中创建任务
```
/task 创建 下午四点开会 16:00 --reminder 15
```

### 群聊中分配任务
@ClawdBot /task 分配 @张三 下午四点开会 16:00

### 查看任务列表
```
/task 列表
```

## 注意事项

- 截止时间格式：`HH:MM` 或 `YYYY-MM-DD HH:MM`
- 提醒提前分钟数：数字（分钟）
- 任务ID 在创建时返回
- 群聊中分配任务需要先获取群成员列表
